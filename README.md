# rewards
Automatização das pesquisas do https://rewards.bing.com/?form=dash_2

## Como utilizar

1. Instale Python no seu PC
2. Instale o VS Code no seu PC
3. Instale o Git Hub no PC
4. Abra o VS Code
5. Utilize o atalho Ctrl + K, Ctrl + O (Esse comando vai abrir o explorador de arquivos)
6. Escolha a pasta onde está o seu arquivo
7. Abra o `pesquisas.txt` e coloque as pesquisas que deseja realizar (Não utilize acentos ou carateres especicias)
8. Salve esse arquivo com o comando Ctrl + S
9. Utilize o atalho Ctrl + '' (Esse botão fica abaixo do ESC no teclado, se o seu teclado for <=70% você vai utilizar com a tecla FN para ativar)
![Botão no teclado](https://aprendelibvrefiles.blob.core.windows.net/aprendelibvre-container/course/informatica_basica/image/tecladoportug_xl.png)
10. Após executar o passo anterior, seu terminal irá abrir
11. Basta digitar, ou colar, o seguinte comando: `python rewards.py`

FIM

Uma tela irá abrir e você pode iniciar a pesquisa
